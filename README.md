# Java-Util
--
本项目是一个 Java 常用的工具类，方便其他开发者使用，提高编码效率。
---
+ DateTimeUtil          日期处理
+ CollectionUtils		集合处理
+ FormateUtil			格式（钱、里程）转换
+ HttpUtil				HTTP客户端
+ JsonUtils 			Json转换
+ MD5Utils 				MD5加密
+ RegexUtil				正则表达式（身份证、邮箱等）
+ SHAUtils				SHA 加密
+ SnowflakeIdUtil		分布式ID随机生产
+ StringUtil			字符串处理
+ UUIDUtil				ID 生产
+ Page					分页工具类
